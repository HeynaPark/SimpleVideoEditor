# SimpleVideoEditor



## Dependency

- PyQt5
- ffmpeg


## What is
This is a tool to easily change video properties.
- Change Video Size to 960x540 or 1280*720 or 1920x1080 or 3840x2160
- Change Video Codec to AVC1
- Cut Video time
- Print Frame numbers on video
